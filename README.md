## twitch-hosting-list

### Installation

1. Install Node

1. `npm -g install gulp-cli`

1. from project root:

  1. `npm install`

  1. `node index.js`

### Configuration

In `index.js`, replce `userId` with your twitch user id.

In `index.js`, replace `'host-list.txt'` with out file path.

In `gulpfile.js`, in 'clean' task, replace `'host-list.txt'` with out file path.
